Projet Python

Description du projet

Ce projet Python consistait à développer une application de gestion de tâches. J'ai utilisé Flask pour créer une API RESTful qui permet de gérer les tâches (ajout, suppression, modification) via des requêtes HTTP. SQLAlchemy a été utilisé pour gérer la base de données SQLite où les tâches sont stockées.

Technologies utilisées

  • Python (version 3.9)
  • Flask (version 2.0.1)
  • SQLAlchemy (version 1.4)
  • SQLite (base de données légère pour prototypage rapide)
